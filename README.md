# HateIP Server

A tool for who hate enter ip everytime - server.

## Install

```
npm install hateip-server -g
```

or

```
npm install hateip-server
```

## Usage

- with bin  
  
```
hateip-server -t "xxxxx,xxxxxxxxxx" -p "9999" -d "localhost"  
```

or  


```
./bin/server -t "xxxxx,xxxxxxxxxx" -p "9999" -d "localhost"  
```
